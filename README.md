# WenDataStudio
老师为雯雯同学做的数据处理项目
-----------
个性化表格数据处理与可视化工具

## 环境要求
- Python 3.9 或更高版本
- pip 包管理工具

## 快速开始

### 安装步骤
```bash
# 克隆仓库
git clone https://github.com/sonw7/WenDataStudio.git
cd WenDataStudio

# 创建并激活虚拟环境
# Windows:
python -m venv venv
venv\Scripts\activate

# macOS/Linux:
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
